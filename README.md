# Custom WordPress Blocks (Gutenberg)

This repository contains custom Gutenberg blocks and a block-based WordPress theme
built with modern WordPress (Block Theme + Create Block).

## Contents
- Custom Gutenberg blocks (dynamic blocks)
- Block-based WordPress theme (Twenty Twenty-Five based)

## Local Development
This project is developed locally using WordPress and Node.js.

Blocks are built using:
- @wordpress/create-block
- npm scripts for development and build

## Notes
- `node_modules` and build artifacts are excluded via `.gitignore`
- This repository contains source code only
